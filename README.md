# 프로젝트 개요

- CSV 파일을 Hex 파일로 변환하는 프로그램이 필요함.

# 프로젝트 목적

- CSV 파일을 `.MBC` 파일로 변환 후 다운로드가 가능하도록 하는 것.
- HxD프로그램으로 열었을 때 열려야 함

# 프로젝트 사용 기술 스택

- React, ElectronJS
